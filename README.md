# 🚀 Node.js & React Project

A professional full-stack web application built with a **Node.js/Express** backend and a **React** frontend. The project features a clean, scalable architectural structure with a clear separation of concerns between client and server.

## 📁 Project Structure

* 💻 **`client/`** — The frontend application and user interface built with React.
* ⚙️ **`server/`** — The backend REST API and business logic built with Node.js.

---

## 🛠️ Technologies Used

### 🎨 Frontend
* **React.js** (Component-based UI)
* **HTML5 & CSS3** (Responsive modern styling)
* **Axios** (Promise-based HTTP client for API requests)

### 🔌 Backend
* **Node.js** (JavaScript runtime environment)
* **Express.js** (Minimalist web framework for APIs)
* **Cors & Dotenv** (Security headers & environment configuration)

## 🏃‍♂️ Getting Started

Follow these simple steps to clone and run the project locally on your machine.

### 📋 Prerequisites
Ensure you have **Node.js** and **npm** installed globally.

### 🏠 1. Setup the Backend (Server)
1. Open your terminal and navigate to the server folder:
   **cd server**
2. Install all required backend packages:
   **npm install**
3. Spin up the development server:
   **npm start**

### 💻 2. Setup the Frontend (Client)
1. Open a new terminal branch and navigate to the client folder:
   **cd client**
2. Install all required frontend packages:
   **npm install**
3. Start the local React server:
   **npm start**

---
